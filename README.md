# re-svg

A command-line tool for processing and manipulating SVG files.

## Installation

Install globally:

```bash
# Using Bun (recommended)
bun add -g re-svg

# Using npm
npm install -g re-svg

# Using yarn
yarn global add re-svg
```

## Development

- Install dependencies:

```bash
bun install
```

- Run development mode with watch:

```bash
bun run dev
```

- Run the unit tests:

```bash
bun run test
```

- Build the library:

```bash
bun run build
```

- Type checking:

```bash
bun run typecheck
```

- Release a new version:

```bash
bun run release
```
